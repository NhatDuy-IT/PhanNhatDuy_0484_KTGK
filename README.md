# PhanNhatDuy_0484_KTGK

## Giới thiệu

Đây là dự án web đơn giản **To-do List** được xây dựng bằng:

- HTML
- CSS
- JavaScript

Ứng dụng cho phép:

- Thêm công việc mới
- Đánh dấu công việc đã hoàn thành
- Xóa công việc
- Lọc theo trạng thái: Tất cả / Chưa hoàn thành / Đã hoàn thành
- Lưu dữ liệu bằng `localStorage`

## Cách chạy dự án

1. Mở file `index.html` bằng trình duyệt
2. Hoặc dùng VS Code với Live Server để chạy

## Cấu trúc thư mục

- `index.html`: Giao diện chính
- `styles.css`: Giao diện CSS
- `script.js`: Xử lý logic ứng dụng

## Mục tiêu bài tập GitHub

Bạn có thể dùng dự án này để luyện các thao tác Git/GitHub như:

- `git init`
- `git add .`
- `git commit -m "Initial commit"`
- tạo repository trên GitHub
- `git push`
